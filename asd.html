One app, three ways to work

Seamlessly switch between coding, version control, and AI assistance without leaving your workspace.

💻

Code Mode

A full VS Code-style editor with real extensions. Multi-pane Monaco editor, quick-open, integrated terminal, and a sandboxed extension host.

  • Real extension host — LSP, debugging, tasks
  • TextMate grammars & themes
  • Open any local folder
🌳

Git Mode

An animated, real-time repo tree that turns branches and merges into something you watch happen, not infer from logs.

  • Live filesystem-watcher-driven tree
  • VS Code-style 3-way merge editor
  • One-click repo cloning
🤖

Agent Mode

Run multiple Claude Code sessions side by side with live token tracking and file-edit hooks.

  • Up to 3 concurrent sessions
  • Live token usage tracking
  • Animated activity indicators

  • Multi-pane Monaco editor with quick-open
  • Real extension host — LSP, debugging, tasks, SCM

Everything else, built in

The features that support all three modes and make Code Companion feel like an app, not a tool.

🔔

Webhook Push Notifications

Get a native desktop alert the instant a watched branch updates, with the author and commit message ready to read.

🔄

Pull Requests & Teams

Browse and open PRs, manage collaborators and teams, all without leaving the app.

✨

AI-Assisted Commits

Get a Gemini-powered commit message drafted straight from your staged changes — ready to edit, or send as-is.

🔐

Seamless GitHub Sign-In

Device-flow authentication keeps your GitHub identity in sync automatically — no manual setup, no copy-pasted tokens.

⬆️

Auto-Updates

Code Companion checks for and installs new releases in the background, so you're never running a stale build.

🌧️

Tray, Launch & Ambience

Lives in your system tray with launch-on-login, plus optional rain and snow effects for the workspace.

Enterprise-grade security, locally

Your code and credentials never leave your machine without your permission. Multiple layers of defense keep your secrets safe.

🔒

Native Credential Vault

Before a key ever touches your disk, it passes through your OS's own secure credential store. Never saved in plain text.

🛡️

Device-Locked Encryption

Military-grade fallback encryption using a unique key derived from your own machine. Copy the file elsewhere — it won't open.

🔍

Pre-Commit Secret Scanning

Every commit is scanned first. If a password or API key slips in, the commit is blocked and flagged before it reaches GitHub.
